# Задание к лабораторной работе №7

## Тема

Изучение систем управления пакетами на примере Hunter.

## Основное задание

1. Создать публичный репозиторий `lab07` на GitHub.
2. Взять проект из лабораторной работы №6.
3. Подключить Hunter к проекту через `HunterGate.cmake`.
4. Заменить локальное или ручное подключение GoogleTest на подключение через
   Hunter.
5. Собрать проект и запустить тесты.
6. Оформить отчет.

## Домашнее задание

Создать свой Hunter-пакет или локальную настройку Hunter-пакета.
