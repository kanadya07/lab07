## Описание

В этой лабораторной работе нужно было познакомиться с системой управления
пакетами Hunter. За основу был взят проект из лабораторной работы №6, после
чего зависимость GoogleTest была подключена через Hunter.

## Что есть в репозитории

- `TASK.md` - задание
- `REPORT.md` - отчет
- `cmake/HunterGate.cmake` - подключение Hunter
- `cmake/Hunter/config.cmake` - локальная настройка Hunter-пакета
- `demo/main.cpp` - демонстрационное приложение
- остальные файлы проекта из прошлых лабораторных
